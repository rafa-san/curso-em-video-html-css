Desafio 6: tags

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobir qual tag deverá utilizar em cada caso. Este desafio só faz sentido em dispositivos que permitem a interação do mouse.

  • Parágrafos
  • Títulos
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas numeradas
  • Listas com demarcadores
